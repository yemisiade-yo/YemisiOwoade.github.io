Newyork Port Authority

I worked on a project that equipped me with the analytical tools and methodologies necessary to tackle complex business challenges, making me well-prepared for a role that requires business, technological, data financial acuity and strategic thinking. It involved creating a Data Warehouse modeling project for the Port Authority of NY and NJ project and forecast analysis till 2030. To do this, I used Excel to consolidate more than 100 (125-150) datasets into one and clean the data received, using SQL server I created corresponding Data warehouse modeling. Then, I identified the historical trends for the number of passengers and buses and projected those trends into the future using excel.

The next step was to build data models and forecasting modeling in R, successfully created and consistently updated R scripts to extract, process, and analyze historical datasets spanning previous years to present, achieving a 95% data completeness rate. With additional factors that might affect passenger numbers such as gas prices, temperatures, sporting events, I created an algorithm using Machine Learning and clustering means. Finally, I communicated my visualization and findings with Microsoft Power BI dashboarding.

POWER BI DASHBOARDS

I Have built this Power BI portfolio to showcase my skills in creating visually compelling dashboards that could help generate business insights, such as KPIs tracking, profit, and loss analysis and Revenue trends, the Executive Summary dashboard offers a comprehensive overview of the company's performance using various visualization charts and tables in Power BI. Leveraging the Adventure Works dataset, I meticulously designed the dashboard to provide actionable insights for both internal stakeholders and external clients.

One of the key features of the dashboard is the Monthly Returns chart, which visually represents the number of product returns over time. Despite the goal being set at 169 returns, the actual number of returns was 166. While this may seem like a deviation from the target, it's essential to note that having fewer returns than anticipated is a positive indicator of customer satisfaction. Hence, the visualization displays a green checkmark to signify this achievement.

Conversely, the Monthly Order chart highlights a different scenario. Here, the goal was set at exceeding 2,165 orders, but the company only managed to achieve 2,146 orders. This shortfall is depicted in red, signaling that the goal wasn't met. This visualization serves as a prompt for further analysis and action to understand why the target wasn't achieved and how to improve order volumes in the future.

In addition to these key insights, the Executive Summary dashboard incorporates a diverse range of visualization charts, including gauges, tables, tree maps, and line charts for forecasting. Each chart is carefully crafted to provide a comprehensive view of the company's performance across various metrics, enabling stakeholders to make informed decisions and drive strategic initiatives. Furthermore, the use of the Adventure Works dataset allows for seamless integration of data from different sources, providing a unified and holistic view of the company's operations. By harnessing the power of Power BI, I've created an interactive and dynamic dashboard that not only tracks performance but also facilitates drill-down analysis and exploration of underlying data trends. In summary, the Executive Summary dashboard represents a culmination of my skills in creating visually compelling and informative dashboards using Power BI. By leveraging the Adventure Works dataset, I've provided stakeholders with a powerful tool for monitoring performance, identifying trends, and driving actionable insights to propel the company forward.
